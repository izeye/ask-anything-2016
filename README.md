# ask-anything
## Overview
* `ask-anything` is an answering framework.

## How To Publish `ask-anything-api` to a Maven repository
* Maven local

```
./gradlew clean publishApiPublicationToMavenLocal
```

* Maven repository

```
./gradlew clean publishApiPublicationToMavenRepository
```
